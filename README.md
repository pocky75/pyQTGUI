# pyQTGUI
UI는 Qt Designer를 통해 제작 하였으며,
PyQt6와 OpenCV를 사용해 웹캠 영상을 실시간으로 보여주고 이름, 전화번호, 메모를 입력받아 CSV 파일로 저장하는 프로그램을 만들었습니다.

# Opencv 모듈 설치
```
pip install opencv-python
```
